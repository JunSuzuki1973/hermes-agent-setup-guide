## 概要

Hermes Agentは、Telegramボットを通じてAIアシスタントと対話するためのツールです。この解説書では、以下の設定について説明します：

- Hermes Agentのインストール
- Kilo Codeプロバイダ設定
- Telegramボット設定
- Gatewayのsystemdサービス設定
- モデル設定の変更
- OpenClawのフォールバック設定

## 必要なもの

- Linuxサーバー（Ubuntu推奨）
- Node.js（v18以降）
- GitHubアカウント
- Telegramアカウント
